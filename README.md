# Suma de Arreglos Paralela con OpenMP (C++)

Este proyecto realiza la suma elemento a elemento de dos arreglos A y B para obtener C:
C[i] = A[i] + B[i]

## Requisitos
- Visual Studio Community (C++)
- OpenMP habilitado: Project Properties → C/C++ → Language → OpenMP Support = Yes (/openmp)

## Cómo ejecutar
1. Compilar y ejecutar el proyecto.
2. Indicar la cantidad de hilos (ej. 2, 4, 8).
3. El programa imprime los primeros 10 elementos de A, B y C y muestra "Validacion: OK".
